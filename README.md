# Music Player

A local music player for Android.

## Features

- Plays audio files stored on your device
- Reads and displays cover art from embedded MP3 metadata
- Lock screen and notification playback controls via Media3

## Build

1. Clone the repo
2. Open in Android Studio
3. Run on a device or emulator (API 26+)

## Libraries Used

- [Media3 / ExoPlayer](https://github.com/androidx/media) — Apache 2.0
- [Coil](https://github.com/coil-kt/coil) — Apache 2.0
- [Room](https://developer.android.com/jetpack/androidx/releases/room) — Apache 2.0

## Disclaimer

This app only reads audio files and metadata (including embedded cover art) stored locally on the user's own device. It does not download, upload, or redistribute any copyrighted content.
